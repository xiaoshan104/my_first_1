1. npm == node package manager

2. npm随着node已经安装完毕。
3. 如果更换npm的下载源服务器的地址        
    a:下载nrm，他可以去自由的更换npm的下载的服务器地址
     npm install -g nrm@latest
    
4. n包 专门用来管理node的版本         
    下载n包：npm install -g n@latest   //目前不支持window
    
用express开发web：
1. 配置package.json
2. 用终端进入当前的目录(package.json所在的目录)

脚手架(cli)：
1. 安装脚手架工具
    npm install -g express-generator@4
2. 进入到项目目录
    express --view=ejs
3. npm install
    
4. 启动服务器
    npm start
